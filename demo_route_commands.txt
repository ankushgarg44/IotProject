Indoor route demo for Firebase realtime testing

This project now builds the demo script from the actual destination route calculated by OSRM.

Flow:
1. Enter destination in lat,lon format.
2. Tap Start Navigation.
3. Route steps and turn directions are loaded into route demo.
4. Tap RUN LOADED ROUTE DEMO.
5. App replays each route command to Firebase with timed delays.

Firebase path used by the app:
bike_commands/direction

Notes:
- Commands are generated from route maneuvers (LEFT, RIGHT, UTURN, STRAIGHT).
- Delay per command is estimated from each route step distance.
- This verifies app -> Firebase -> ESP32 behavior without moving outside.
